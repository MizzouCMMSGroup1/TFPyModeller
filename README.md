Requirements:
=============
#####Required:  
* [Python 2.7 32 bit](http://www.python.org/download/)  
* [dDFIRE](http://sparks.informatics.iupui.edu/yueyang/server/dDFIRE/)
* Lipa
* Rama
* Crankite

#####Optional:  
* [Jmol](http://jmol.sourceforge.net/download/)
